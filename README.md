# Event Ticketing Platform

A lightweight, minimalistic college event ticketing platform. 
Users can browse events and register. Admins can manage events and validate attendees via QR code or registration ID.

## Features
- **Visitor:** Browse events and register (no login required).
- **Admin:** Full dashboard to manage events, view registrations, and verify entry via QR code scanning.
- **Tech Stack:** React (Vite), Tailwind CSS, Netlify Functions, TiDB Cloud.

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Setup environment variables for database connections

Built for college use. Keep it simple, keep it clean. 🚀
